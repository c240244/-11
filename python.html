Lesson 1: Introduction to Python

What is Python?

Python is a high-level, interpreted programming language known for its clear syntax and readability. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages in the world.

Did You Know?

Python was named after the British comedy group Monty Python, not the snake!

Why Learn Python?

Python is versatile and can be used for:

  • Web development (Django, Flask)
  • Data analysis and visualization (Pandas, Matplotlib)
  • Machine learning and AI (TensorFlow, PyTorch)
  • Automation and scripting
  • Game development (Pygame)
  • Scientific computing

Your First Python Program

Let's start with the traditional "Hello, World!" program. In Python, this is very simple:

# This is a comment
print("Hello, World!")
# Output: Hello, World!

To run this code, you can save it in a file with a .py extension (e.g., hello.py) and run it using Python interpreter, or type it directly into the Python shell.

Python Syntax Basics

Python uses indentation to define code blocks, unlike other languages that use braces {}. This makes Python code very readable.

# If statement example
if 5 2:
    print("Five is greater than two!")
# Output: Five is greater than two!

Note how the print statement is indented. This indentation is required for the code to work properly.

✏️Try It Yourself

Open a Python interpreter or create a Python file and try the following:

  1. Print your name using the print() function
  2. Write an if statement that checks if 10 is greater than 5 and prints a message
  3. Try writing code without proper indentation to see what error you get

Python Features

Python has several features that make it a great choice for beginners and experts alike:

  • Easy to Learn: Clean and straightforward syntax
  • Interpreted: No need for compilation
  • Cross-platform: Runs on Windows, macOS, Linux
  • Large Standard Library: Comes with many built-in modules
  • Community Support: Active community with extensive documentation

Setting Up Python

To get started with Python, you'll need to install it on your computer:

  1. Download the latest version from the official Python website
  2. Run the installer and follow the instructions
  3. Verify the installation by opening a terminal and typing python --version

You might also want to install an Integrated Development Environment (IDE) like PyCharm or Visual Studio Code to make coding easier.

Learning Resources

  • Official Python Documentation: docs.python.org
  • Python for Beginners: python.org/about/gettingstarted
  • Codecademy Python Course: codecademy.com/learn/learn-python

If you're new to programming, don't worry about understanding everything at once. Focus on the basics and practice regularly. Programming is a skill that improves with time and experience.

Lesson 2: Variables & Data Types

What are Variables?

Variables are used to store data values. In Python, you don't need to declare a variable with a specific type. The type is inferred from the value you assign.

# Variable assignment
x = 5
name = "John"
is_student = True

# Print variables
print(x)
print(name)
print(is_student)

Basic Data Types

Python has several built-in data types:

  • Integers: Whole numbers (e.g., 5, -3, 1000)
  • Floats: Decimal numbers (e.g., 3.14, -0.5, 2.0)
  • Strings: Text data (e.g., "hello", 'Python')
  • Booleans: True or False values
  • None: Represents absence of value
# Examples of different data types
age = 25 # Integer
height = 5.9 # Float
name = "Alice" # String
is_adult = True # Boolean
score = None # None

Type Conversion

You can convert between different data types using built-in functions:

# Type conversion examples
x = 10
y = 3.14
z = "20"

# Convert to float
float_x = float(x) # 10.0

# Convert to integer
int_y = int(y) # 3

# Convert to string
str_x = str(x) # "10"

# Convert string to integer
int_z = int(z) # 20

Practice Exercise

Try these exercises to practice working with variables:

  1. Create variables to store your name, age, and favorite color
  2. Print a message that includes all these variables
  3. Convert a string containing a number to an integer and perform arithmetic operations with it
  4. Create a variable with value None and check its type using the type() function

Variable Naming Rules

When naming variables in Python, follow these rules:

  • Variable names can contain letters, numbers, and underscores
  • They cannot start with a number
  • They are case-sensitive (age, Age, and AGE are different variables)
  • Use descriptive names that indicate the purpose of the variable
  • Follow the snake_case naming convention (e.g., user_name, is_logged_in)

Best Practice

Always use descriptive variable names. Instead of short names like 'x' or 'n', use names that describe what the variable represents, like 'student_count' or 'is_authenticated'. This makes your code much more readable.

Lesson 3: Operators & Expressions

Arithmetic Operators

Python supports various arithmetic operators for mathematical operations:

# Arithmetic operators
x = 10
y = 3

print(x + y) # Addition: 13
print(x - y) # Subtraction: 7
print(x * y) # Multiplication: 30
print(x / y) # Division: 3.333...
print(x // y) # Floor division: 3
print(x % y) # Modulus: 1
print(x ** y) # Exponentiation: 1000
