Lesson 1: Introduction to Java

What is Java?

Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It was developed by Sun Microsystems (now owned by Oracle) and first released in 1995.

Did You Know?

Java was originally called Oak after an oak tree that stood outside James Gosling's office. Later it was renamed to Java, from Java coffee.

Why Learn Java?

Java is versatile and widely used for:

  • Enterprise applications
  • Android app development
  • Web applications
  • Scientific computing
  • Big data technologies (Hadoop, Spark)
  • Game development

Your First Java Program

Let's start with the traditional "Hello, World!" program. In Java, this requires a class and a main method:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

To run this code, you need to save it in a file named HelloWorld.java, compile it with javac HelloWorld.java, and then run it with java HelloWorld.

Java Syntax Basics

Java uses curly braces {} to define code blocks, and semicolons to end statements. It's a statically-typed language, meaning you must declare the type of variables.

// If statement example
int number = 5;
if (number 2) {
    System.out.println("Five is greater than two!");
}

Try It Yourself

Set up Java on your computer and try the following:

  1. Create and run the HelloWorld program
  2. Modify it to print your name
  3. Create a program that declares an integer variable and prints it

Java Features

Java has several features that make it a great choice for development:

  • Platform Independent: Write once, run anywhere (WORA)
  • Object-Oriented: Follows OOP principles
  • Robust: Strong memory management and exception handling
  • Secure: Provides security features like bytecode verification
  • Multithreading: Built-in support for concurrent programming

Setting Up Java

To get started with Java, you'll need to install the Java Development Kit (JDK):

  1. Download the latest JDK from the official Oracle website
  2. Run the installer and follow the instructions
  3. Set the JAVA_HOME environment variable
  4. Add Java to your system PATH
  5. Verify the installation by opening a terminal and typing java -version

You might also want to install an Integrated Development Environment (IDE) like IntelliJ IDEA, Eclipse, or NetBeans to make coding easier.

Learning Resources

  • Official Java Documentation: docs.oracle.com/javase
  • Java Tutorials: docs.oracle.com/javase/tutorial
  • Codecademy Java Course: codecademy.com/learn/learn-java

Note for Beginners

If you're new to programming, Java might seem more complex than some other languages due to its verbose syntax. Don't worry - as you practice, you'll become comfortable with the structure and conventions.

Lesson 2: Variables & Data Types

What are Variables?

Variables are containers for storing data values. In Java, you must declare a variable with a specific type before you can use it.

// Variable declaration and assignment
int age = 25;
double height = 5.9;
char grade = 'A';
boolean isJavaFun = true;
String name = "John";

// Print variables
System.out.println(age);
System.out.println(name);

Primitive Data Types

Java has 8 primitive data types:

  • byte: 8-bit integer (-128 to 127)
  • short: 16-bit integer (-32,768 to 32,767)
  • int: 32-bit integer (most commonly used)
  • long: 64-bit integer
  • float: Single-precision 32-bit floating point
  • double: Double-precision 64-bit floating point (most commonly used)
  • boolean: true or false values
  • char: Single 16-bit Unicode character
// Examples of primitive data types
byte smallNumber = 100;
short mediumNumber = 10000;
int largeNumber = 100000;
long veryLargeNumber = 1000000000L;
float price = 10.99f;
double preciseValue = 3.1415926535;
boolean isSunny = true;
char firstLetter = 'J';

Reference Data Types

In addition to primitives, Java has reference types like:

  • String: For text data (though technically a class)
  • Arrays: For storing collections of data
  • Classes: Custom data types you define
  • Interfaces: Abstract types that define behavior
// String examples
String greeting = "Hello, Java!";
String name = "Alice";

// Array examples
int[] numbers = {1, 2, 3, 4, 5};
String[] colors = {"red", "green", "blue"};

Type Conversion

Java supports both implicit (widening) and explicit (narrowing) type conversion:

// Implicit conversion (widening)
int myInt = 10;
double myDouble = myInt; // Automatically converts int to double

// Explicit conversion (narrowing)
double anotherDouble = 9.78;
int anotherInt = (int) anotherDouble; // Manual casting: double to int

System.out.println(myDouble); // Outputs 10.0
System.out.println(anotherInt); // Outputs 9 (decimal part lost)

Practice Exercise

Try these exercises to practice working with variables:

  1. Create variables to store your name, age, and favorite programming language
  2. Print a message that includes all these variables
  3. Create an array of your top 3 favorite foods and print them
  4. Try converting between different data types (int to double, double to int, etc.)

Variable Naming Rules

When naming variables in Java, follow these rules:

  • Variable names can contain letters, digits, underscores, and dollar signs
  • They must begin with a letter, $ or _
  • They are case-sensitive (age, Age, and AGE are different variables)
  • Use descriptive names that indicate the purpose of the variable
  • Follow the camelCase naming convention (e.g., userName, isLoggedIn)
  • Avoid using Java keywords (like int, class, public, etc.)

Best Practice

Always use descriptive variable names. Instead of short names like 'x' or 'n', use names that describe what the variable represents, like 'studentCount' or 'isAuthenticated'. This makes your code much more readable.

Lesson 3: Operators & Expressions

Arithmetic Operators

Java supports various arithmetic operators for mathematical operations:

// Arithmetic operators
int x = 10;
int y = 3;

System.out.println(x + y); // Addition: 13
System.out.println(x - y); // Subtraction: 7
System.out.println(x * y); // Multiplication: 30
System.out.println(x / y); // Division: 3
System.out.println(x % y); // Modulus: 1

// Increment and decrement
x++; // x is now 11
y--; // y is now 2
